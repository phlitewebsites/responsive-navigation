# Responsive Navigation

## Usage

```js
var menu = require('responsive-navigation')

// One list
menu('#main-nav')

// or multiple lists
menu(['#pre-nav', '#main-nav', '#post-nav'])
```

#### Browser Support

* Internet Explorer 9+
* Chrome (latest)
* Edge (latest)
* FireFox (latest)
* Safari (latest)
