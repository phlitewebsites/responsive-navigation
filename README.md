# Responsive Navigation

## Usage

```js
var menu = require('responsive-navigation')

// One list
menu('#main-nav')

// or multiple lists
menu(['#pre-nav', '#main-nav', '#post-nav'])
```

#### Browser Support

* Chrome (latest)
* Edge (latest)
* FireFox (latest)
* Safari (latest)
* Internet Explorer 11
